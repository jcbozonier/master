== Description ==

iNove theme inspired by MacZone.sk. It is very stylish, widget supported and doesn’t require any plugin.

Tested on WordPress 2.3.3/2.5.1/2.6.3/2.7, with IE 6/7, Firefox 3, Opera 9.6 and Safari 3.2.1.
Valid XHTML 1.1 and CSS 3!
Best view under 1024×768 or greater.
